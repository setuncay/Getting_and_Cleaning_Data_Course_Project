# Code Book

## Variable list and descriptions

* Subject
    * Identifies the subject who performed the activity for each window sample. Numeric value -  Range [1,30]
* Activity
    * Any of the six activities performed:
        1. WALKING
        2. WALKING_UPSTAIRS
        3. WALKING_DOWNSTAIRS
        4. SITTING
        5. STANDING
        6. LAYING
